How to use?

Include LoadingIndicator and put it somewhere in the top-component. For example: ` import LoadingIndicator from "react-loading-indicator";

var Layout = React.createClass({ render: function() { return (

); } }); `

Now, whenever you need to show an indicator, trigger loader.show, for example: ` loadPage: function() { $(window).trigger("loader.show"); // do your ajax thing. },

loadPageCallback: function() { $(window).trigger("loader.hide"); } `
